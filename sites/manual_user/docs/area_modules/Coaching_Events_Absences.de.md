## Als Betreuer - Als Klassenlehrer {: #tabs_coach-master_coach}

![coaching_events_absences_events_coach-master_v1_de.png](assets/coaching_events_absences_events_coach-master_v1_de.png){ class="shadow lightbox" }

Auf oberster Ebene können Sie die Ansicht Ihrer Termine und Absenzen fokussieren auf Ihre Rolle als Betreuer:in oder ggf. als Klassenlehrer:in zu fokussieren.

Im [Course Planner](../../manual_user/area_modules/Course_Planner.de.md) besteht die Möglichkeit, einer Person die Rolle der Klassenlehrerin bzw. des Klassenlehrers zuzuweisen. 
(Voraussetzung: Das Modul Course Planner wurde von einem/einer Administrator:in aktiviert.)
Für jede Durchführung kann dies eine andere Person sein. In diesem Fall sind dann im Coachingtool die Termine und Absenzen Ihrer Klassen angezeigt.

## Tab Cockpit {: #tab_cockpit}

Im Tab Cockpit finden Sie eine Tagesübersicht mit den Abschnitten 

* Termine
* Absenzen
* Meldungen

(Je nach voher getroffenem Fokus für Sie als Betreuer:in oder als Klassenlehrer:in.)<br>
Anstatt des aktuellen Tages kann auch ein anderer Tag ausgewählt werden.

![coaching_events_absences_tab_events_cockpit_v1_de.png](assets/coaching_events_absences_tab_events_cockpit_v1_de.png){ class="shadow lightbox" }

Durch Klick auf das **Buchsymbol** in der Zeile eines Termins können Absenzen erfasst werden.

Durch Klick auf das **Symbol mit den 3 Punkten** in der Zeile eines Termins können einzelne Termine als Excel-Datei exportiert, eine Absenzenliste und eine Präsenzliste im PDF-Format heruntergeladen werden.

## Tab Termine {: #tab_events}

Um den Überblick über viele Termine zu behalten, stehen Ihnen mehrere Filterfunktionen zur Verfügung.

![coaching_events_absences_tab_events_events_v1_de.png](assets/coaching_events_absences_tab_events_events_v1_de.png){ class="shadow lightbox" }

Rechts oben über der Liste lassen sich mit dem Zahnrad-Icon die gewünschten Spalten auswählen. Passen Sie die Listenansicht Ihren Bedürfnissen an.

**Beispiele:**

* Durch Klick auf das **Buchsymbol** in der Zeile eines Termins können Absenzen erfasst werden.

* Die Spalte mit dem **Stern-Symbol (Asterisk)** zeigt Ihnen, ob eine Anwesenheit verpflichtend geplant wurde oder nicht.

* Durch Klick auf das **Symbol mit den 3 Punkten** in der Zeile eines Termins können einzelne Termine als Excel-Datei exportiert, eine Absenzenliste und eine Präsenzliste im PDF-Format heruntergeladen werden.

Durch Klick auf das Plus-Symbol am Zeilenanfang können Sie die Details dieses Termins aufklappen.

## Tab Absenzen {: #tab_absences}

Das Erfassen von Absenzen kann in den Tabs **Cockpit** oder **Termine** durch Klick auf das **Buchsymbol** in der Zeile eines Termins gemacht werden.

Im Tab Absenzen erhalten Sie dann einen Überblick über die erfassten Absenzen. 

* Im Suchfeld können Sie nach Benutzer:innen, Dozent:innen, Kurstiteln und Terminen suchen.
* Schränken Sie den Zeitraum ein, für den Absenzen gesucht werden sollen.
* In einer Vorauswahl können Sie alle oder nur die unentschuldigten Absenzen anzeigen lassen.

![coaching_events_absences_tab_absences1_v1_de.png](assets/coaching_events_absences_tab_absences1_v1_de.png){ class="shadow lightbox" }

Um eine Absenz zu entschuldigen, markieren Sie die betreffende Person. Sobald mindestens eine Person markiert ist, erscheint der Button zum Erfassen einer Entschuldigung über der Liste.

![coaching_events_absences_authorize_v1_de.png](assets/coaching_events_absences_authorize_v1_de.png){ class="shadow lightbox" }

## Tab Meldungen {: #tab_notices}

Alle Meldungen im Zusammenhang mit Absenz und Dispens sind in diesem Tab gesammelt und können Ihren Bedürfnissen entsprechend gefiltert angezeigt werden.

![coaching_events_absences_tab_notices_v1_de.png](assets/coaching_events_absences_tab_notices_v1_de.png){ class="shadow lightbox" }

Im **Suchfeld** können Sie nach Benutzer:innen, Dozent:innen, Kurstiteln und Terminen suchen.

Filtern Sie nach **Art der Meldung** ("Ohne Abmeldung", "abgemeldet", "dispensiert") oder<br> nach **entschuldigt** und/oder **unentschuldigt**.

Als Filteroption **"Grund"** stehen Ihnen die vom Administrator vordefinierten Begriffe zur Auswahl.

Ausserdem finden Sie hier Buttons für

* Neue Absenzmeldung erfassen
* Neue Dispens erfassen
* Abmeldung

## Tab Rekurse {: #tab_appeals}

In diesem Tab verschaffen Sie sich einen Überblick für die Verwaltung der aller Rekurse.<br>

Im **Suchfeld** können Sie nach Benutzer:innen, Betreuer:innen und Terminen suchen.

Filtern Sie die Liste z.B. nach dem **Status** "pendent", "abgelehnt" oder "angenommen". (Sie können wahlweise einen Status, zwei davon oder alle auswählen.)

![coaching_events_absences_tab_appeals_v1_de.png](assets/coaching_events_absences_tab_appeals_v1_de.png){ class="shadow lightbox" }

## Tab Personensuche {: #tab_user_search}

Verwenden Sie für die Personensuche das Suchfeld und die Möglichkeit zum Umschalten der Vorauswahl (Suche nach Kurs, Suche nach Teilnehmer:innen, Suche nach Produkt).

Wenn Sie auf einen der gefundenen Kurse oder ein Produkt klicken, gelangen Sie zur Liste der dort zugeordneten Personen.
