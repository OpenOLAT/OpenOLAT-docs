# Coaching - Reports {: #reports}


![coaching_reports1_v1_de.png](assets/coaching_reports1_v1_de.png){ class="shadow lightbox" }


Für die Erstellung von Reports stehen verschiedene Vorlagen zur Verfügung, mit denen Sie sich einen Überblick über die ausgestellten Zertifikate verschaffen können.

Durch Klick auf die Pfeile in der Spalte "Ausführen" werden anhand dieser Vorlagen Excel-Dateien mit den aktuellen Daten erzeugt.

 ![coaching_reports2_v1_de.png](assets/coaching_reports2_v1_de.png){ class="shadow lightbox" }

Die so erstellten Excel-Dateien finden Sie im unteren Bereich des Screens alle aufgelistet. Sie können kopiert und heruntergeladen werden.

 ![coaching_reports3_v1_de.png](assets/coaching_reports3_v1_de.png){ class="shadow lightbox" }
