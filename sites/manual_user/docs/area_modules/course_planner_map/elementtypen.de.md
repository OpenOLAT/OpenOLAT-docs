# Strukturregeln festlegen (Elementtypen)

[:material-arrow-left: Zurück zur Übersicht](../../Course_Planner_Map/)

Bevor das erste Bildungsprodukt entsteht, legt die Administration die Spielregeln fest: Aus welchen Bausteinen darf ein Lehrgang bestehen, was darf zuoberst stehen, was worin liegen? In OpenOlat heissen diese Regeln «Elementtypen»; sie geben allen späteren Strukturen einen verlässlichen Rahmen.

## Wie setze ich das um?

Über «Neuen Typ hinzufügen» einen Elementtyp mit Titel und Kennzeichen anlegen und unter «Verwendung als» festlegen, ob er als Durchführung (oberstes Elternelement) oder als Element dient. Optional «Mit Kursinhalt», «Als Durchführung erlauben» und Features (Absenz, Stundenplan, Fortschritt) konfigurieren.

## Vorbedingungen

Course Planner muss aktiviert sein. Für das Feature Absenzenmanagement muss das Modul Absenzenverwaltung aktiv sein.

## Wo finde ich die Einstellung

Administration › Module › Course Planner › Tab Elementtypen

## Zusammenhänge

Elementtypen bestimmen, welche Elemente ein Produkt enthalten darf und ihre Hierarchie (z. B. Lehrgang > Semester > Modul > Kurs). Pro Typ lassen sich Automatisierungsregeln als Vorlage hinterlegen, die einzelne Elemente übernehmen oder überschreiben.
