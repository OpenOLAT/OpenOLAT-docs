# Mitglieder verwalten

!!! warning "Konzeptstudie: mögliche neue Darstellung"
    Diese Seite ist Teil eines Versuchs, wie der visuelle Einstieg in den Course Planner künftig aussehen könnte. Die Inhalte sind bewusst verkürzt. Verbindlich ist die reguläre Handbuchseite [Course Planner](../../Course_Planner/).

[:material-arrow-left: Zurück zur Übersicht](../../Course_Planner_Map/)

Teilnehmende durchlaufen einen nachvollziehbaren Status-Workflow: ausstehend, aktiv, abgelehnt, entfernt. Wizards führen durch das Hinzufügen (inklusive Rechnungsadresse), Massenaktionen und die Mitgliedschaftshistorie halten auch grosse Gruppen im Griff.

## Wie setze ich das um?

Im Tab «Mitglieder» einer Durchführung über «Teilnehmer:innen hinzufügen» Personen zuweisen. Je nach Reservierungspflicht bestätigen die Eingeladenen ihre Mitgliedschaft; optional muss ein Buchungswunsch von Linienvorgesetzten/Ausbildungsverantwortlichen genehmigt werden.

## Vorbedingungen

Durchführung vorhanden; die eingebundenen Kurse sind auf «Einbindung in Produkt» gesetzt (keine eigene Kurs-Mitgliederverwaltung).

## Wo finde ich die Einstellung

Course Planner › Durchführung › Tab Mitglieder

## Zusammenhänge

Die Mitgliedschaft besteht auf Durchführungsebene, nicht auf Kurs- oder Produktebene. Zum Produkt können nur Besitzer:innen, keine Teilnehmer:innen hinzugefügt werden. Die Zuweisung löst eine Systembenachrichtigung per E-Mail aus.
