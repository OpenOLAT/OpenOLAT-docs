# Besitzer/in

!!! warning "Konzeptstudie: mögliche neue Darstellung"
    Diese Seite ist Teil eines Versuchs, wie der visuelle Einstieg in den Course Planner künftig aussehen könnte. Die Inhalte sind bewusst verkürzt. Verbindlich ist die reguläre Handbuchseite [Course Planner](../../Course_Planner/).

[:material-arrow-left: Zurück zur Übersicht](../../Course_Planner_Map/)

Besitzer/in einer Lernressource oder eines Kurses. Bei im Course Planner eingebundenen Kursen eingeschränkt: Die Mitgliederverwaltung läuft über den Course Planner, To-dos können als erledigt markiert werden.

## Zugriff

Vollzugriff (lesend und schreibend) auf den eingebundenen Kurs; die Mitgliederverwaltung läuft jedoch über die Durchführung, nicht im Kurs.
