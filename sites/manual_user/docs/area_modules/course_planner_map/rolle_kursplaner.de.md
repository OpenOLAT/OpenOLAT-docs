# Kursplaner/in

!!! warning "Konzeptstudie: mögliche neue Darstellung"
    Diese Seite ist Teil eines Versuchs, wie der visuelle Einstieg in den Course Planner künftig aussehen könnte. Die Inhalte sind bewusst verkürzt. Verbindlich ist die reguläre Handbuchseite [Course Planner](../../Course_Planner/).

[:material-arrow-left: Zurück zur Übersicht](../../Course_Planner_Map/)

Organisationsrolle mit Vollzugriff auf den Course Planner: verwaltet Produkte und Durchführungen der eigenen Organisation, nutzt Import und Export und öffnet eingebundene Kurse im Lesemodus mit Statusänderungsrecht.

## Zugriff

Organisationsrolle mit vollem (schreibendem) Zugriff auf den gesamten Course Planner; bei Zuordnung zu einer Organisationseinheit auf deren Course Planner beschränkt.
