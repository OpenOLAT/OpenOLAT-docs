# Lehrgang gliedern (Elemente)

!!! warning "Konzeptstudie: mögliche neue Darstellung"

[:material-arrow-left: Zurück zur Übersicht](../../Course_Planner_Map/)

Innerhalb einer Durchführung wird der Lehrgang gegliedert: in Module, Fächer oder Klassen, je nach Strukturregeln beliebig verschachtelt. In OpenOlat heissen diese Gliederungseinheiten «Elemente»: Jede hält ihre Kurse, Termine, Planungsaufgaben, Absenzen und Mitglieder zusammen.

## Wie setze ich das um?

Im Tab «Struktur» einer strukturierten Durchführung über «Erstellen» ein Element des gewünschten Elementtyps hinzufügen. Über die 3 Punkte bearbeiten oder mit «Element verschieben» in der Baumstruktur umordnen (nicht erlaubte Zielpositionen sind ausgegraut).

## Vorbedingungen

Strukturierte Durchführung; in der Administration definierte Elementtypen.

## Wo finde ich die Einstellung

Course Planner › Durchführungen › Durchführung wählen › Tab Struktur

## Zusammenhänge

Elemente gliedern die Durchführung hierarchisch. Elemente «Mit Kursinhalt» tragen Kurse, reine Strukturelemente nicht (vergleichbar mit dem Kursbaustein «Struktur»). Der Elementtyp bestimmt die erlaubten Positionen im Baum.
