# Absenzen erfassen

!!! warning "Konzeptstudie: mögliche neue Darstellung"

[:material-arrow-left: Zurück zur Übersicht](../../Course_Planner_Map/)

Anwesenheiten und Absenzen werden je Durchführung und Element im eigenen Absenzen-Tab erfasst. Die Erfassung auf Produktebene ist geplant.

## Wie setze ich das um?

Absenzen im Tab «Absenzen» einer Durchführung erfassen. Der Tab erscheint erst, wenn die Absenzen auf dem Element aktiviert wurden.

## Vorbedingungen

Absenzenmanagement muss aktiviert und für das Produkt bzw. den Elementtyp freigegeben sein. Aktivierung in Tab Einstellungen › Optionen › Absenzenkonfiguration.

## Wo finde ich die Einstellung

Course Planner › Durchführung › Tab Absenzen (Aktivierung: Tab Einstellungen › Optionen › Absenzenkonfiguration)

## Zusammenhänge

Absenzen werden je Termin erfasst (Kategorien Anwesend, Entschuldigt, Unentschuldigt, Dispensiert). Daraus wird die Anwesenheitsrate berechnet und gegen die erlaubte Absenzquote geprüft, um zu bestimmen, ob ein Kurs als besucht gilt.
