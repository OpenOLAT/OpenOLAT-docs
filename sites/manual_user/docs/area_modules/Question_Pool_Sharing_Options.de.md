# Fragenpool: Freigabemöglichkeiten {: #question_bank_sharing_options}

Es gibt verschiedene Möglichkeiten, um im Fragenpool anderen Personen Zugriff auf persönliche Frage-Items zu geben. Dabei können Sie jeweils auch entscheiden, ob die Personen die Fragen nur ansehen oder auch ändern dürfen.

## a) Einzelne Personen zu Mit-Besitzer:innen einer Frage machen {: #co-owners}

Wird die ausgewählte Person (Mit-)Besitzer:in der Frage, erscheint die Frage bei dieser Person ebenfalls unter "Meine Fragen". Ähnlich wie bei den OpenOlat Lernressourcen, haben Mit-Besitzer:innen sämtliche Berechtigungen für diese Frage.

Gehen Sie in den Bereich "Mein Fragenpool" → "Meine Fragen", wählen Sie die gewünschte/n Frage/n und wählen Sie über den Link "Autorenrechte" die Personen aus, die Sie zu Mit-Besitzer:innen der Frage machen möchten.

## b) Freigabe in einen Pool {: #sharing_in_a_pool}

Fragen können mit weiteren Personen geteilt werden, indem sie einem oder mehreren existierenden Fragenpools hinzugefügt werden. Welche Fragenpools zur Verfügung stehen und gewählt werden können, ist von dem/der Pooladministrator:in festgelegt. Meist existiert zumindest ein öffentlicher Pool.

Gehen Sie in den Bereich "Mein Fragenpool" → "Meine Fragen", wählen Sie die gewünschte/n Frage/n und wählen Sie über den Link "Freigeben" → "In Pool freigeben" den gewünschten Pool aus. Sie haben anschliessend die Möglichkeit zu entscheiden, ob die Frage im freigegebenen Pool auch bearbeitet werden darf oder nicht.

## c) Freigabe für eine Gruppe {: #release_for_a_group}

Sie können Fragen auch einer im Vorfeld angelegten [OpenOlat Gruppe](../groups/index.de.md) zugänglich machen. Erstellen Sie beispielsweise eine Gruppe, wenn Sie mit mehreren Personen an einer Sammlung von Fragen für einen Online-Test arbeiten oder eine gemeinsame Fragenbasis entwickeln wollen. Der Vorteil bei diesem Weg ist, dass Sie die Personen individuell zusammenstellen können und relativ einfach auch später noch weitere Personen der Gruppe hinzufügen können.

Gehen Sie in den Bereich "Mein Fragenpool" → "Meine Fragen", wählen Sie die gewünschte/n Frage/n und wählen Sie über den Link "Freigeben" → "In Gruppe freigeben" die gewünschte Gruppe aus. Sie haben anschliessend die Möglichkeit zu entscheiden, ob die Frage im freigegebenen Pool auch bearbeitet werden darf oder nicht.

Achtung, die geteilten Fragen werden nur referenziert und nicht kopiert! Anschliessende Änderungen erfolgen von allen Berechtigten an derselben Frage. Möchten Sie die Ursprungsfrage behalten, sollten Sie diese vorher kopieren.

Für die gemeinsame Arbeit an Fragen bietet sich auch die Verwendung des [Beurteilungsprozesses](Question_Bank_Review_Process.de.md), sowie die Nutzung des Status an.
