# General Functions: Info Page {: #general_functions_info}

## What's the purpose of the info page? {: #purpose}

Each learning resource has an information page. This page contains information provided by the system, as well as details that the resource owner can add themselves. These are available to interested users as soon as the learning resource is published (regardless of whether they have registered), even before they access the learning resource. This makes sense if you want to inform the target audience in advance, for example, if the course involves a fee.

## How do you access the info page? {: #access}

### Access the info page via the course overview {: #access_course_overview}

In the main navigation, open your course overview by clicking on "Courses."<br>
Then click the link next to the button to open a course.

![general_functions_infopage_access_courses_v1_de.png](assets/general_functions_infopage_access_courses_v1_de.png){ class="lightbox" }

[To the top of the page ^](#general_functions_info)

---


### Access the info page in the catalog {: #access_catalog}

In the catalog's tile view, you'll find the button to view the information page in the bottom right corner. You can also click on the image.<br>

![general_functions_infopage_access_catalog_tile_v1_de.png](assets/general_functions_infopage_access_catalog_tile_v1_de.png){ class="lightbox" }

In the list view, click the "Learn more" link or the title of the learning resource. 

![general_functions_infopage_access_catalog_list_v1_de.png](assets/general_functions_infopage_access_catalog_list_v1_de.png){ class="lightbox" }

[To the top of the page ^](#general_functions_info)

---


### Access the info page withing a course {: #access_within_a_course}

Once you're in the course, select the "Course Info" icon in the toolbar. 

![general_functions_infopage_access_toolbar_v1_de.png](assets/general_functions_infopage_access_toolbar_v1_de.png){ class="lightbox" }

[To the top of the page ^](#general_functions_info)

---


### Go to the information page on other learning resources {: #access_other_learning_resources}

The information page for other learning resources can be accessed just like the information page for the courses.
Unlike courses, it also includes instructions for use.
