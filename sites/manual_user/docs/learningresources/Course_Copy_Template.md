# Save (a course) as template {: #course_copy_template}

With the Course Planner, courses can be planned in advance and the course is integrated into a implementation at a later date. They are instantiated from a course template. 

If you want to create such a template from an existing course, use the **Save as template** option in the corresponding course under **Administration**. The existing course (regardless of its technical type) is then copied in a way that a template is created for instantiation.
