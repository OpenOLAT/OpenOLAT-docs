# Personal Configuration: Settings

![pers_menu_settings_v1_de.png](assets/pers_menu_settings_v1_de.png){ class="aside-right lightbox"}

![icon_settings.png](assets/icon_settings.png)

The settings allow you to adapt OpenOlat according to your needs.



## Tab System {: #tab_system}

![pers_menu_settings_tab_system_v1_de.png](assets/pers_menu_settings_tab_system_v1_de.png){ class=" shadow lightbox" }

### ![1_green_24.png](assets/1_green_24.png) General system settings: {: #general}

Here you can read out your username and your OpenOlat role. You can also select your individual system language. OpenOlat has been translated into numerous languages. Changing the language will only become active after the next login.

!!! warning "Attention"

	The language of course content is not affected by the language selection here.

You can also set how often you want to receive email notifications. You can choose between: switched off, monthly, weekly, daily, half-day, every four and every two hours. You can also specify whether OpenOlat emails should only be displayed in the inbox within the OpenOlat system, or whether they should also
be sent to the external e-mail address configured in your profile. In the notification mail, you will also find a note regarding new emails within the OpenOlat mailbox, if you have selected the option "Deliver emails to the internal OpenOlat mailbox".

Via the "Character set for download" you can determine in which character set files that you download via data archiving should be stored. This concerns the download of test and questionnaire results as well as course results. The default character set is ISO-8859-1. If your tests or questionnaires contain e.g. Arabic characters you have to select the character set UTF-8.


### ![2_green_24.png](assets/2_green_24.png) Specific system settings

In the Special System Settings, you can view and change the settings that can make it easier for you to get started and work with OpenOlat on a daily basis.

**Restore Session**<br>
In the "Restore Session" drop-down menu, you can specify what happens immediately after logging in: Either

  * the starting page is loaded,
  * the location last visited in OpenOlat is loaded automatically, or
  * depending on the situation, you decide where you want to go after logging in.


**Starting page**<br>
If you have selected the "Yes, automatically" setting, the field for entering the home page will be hidden. This field allows you to set any page within OpenOlat as your personal home page, thereby overriding the system-wide home page setting.

You can find page-specific links in the social sharing bar at the bottom left under "Copy link". It's even easier to simply click the homepage icon on the page you want to set as your homepage. Any OpenOlat page you have access to can be selected as your personal homepage this way.

**Homepage icon in the footer:**
![pers_menu_settings_tab_system_pers_startpage_v1_de.png](assets/pers_menu_settings_tab_system_pers_startpage_v1_de.png){ class=" shadow lightbox" }



### ![3_green_24.png](assets/3_green_24.png) Personal tools

Here you select which user tools should appear in the top right-hand corner of the menu bar (next to your profile picture) so that you can access these tools very quickly.<br>
Tools that appear in the Quick Access bar are no longer listed in the personal menu.

**Example: "Settings" and "Badges" have been moved from the personal menu to the Quick Access bar**
![pers_menu_settings_tab_system_pers_tools_v1_de.png](assets/pers_menu_settings_tab_system_pers_tools_v1_de.png){ class=" shadow lightbox" }

!!! tip "Hint"

	Don't try to enable all the tools; instead, select only the ones you use frequently. This will keep the menu bar uncluttered.


### ![4_green_24.png](assets/4_green_24.png) Reset Settings

Under "Reset Settings" in the “System” tab, you can **reset** any system-related changes you've made **back to their default settings**. The following can be reset:

* the personalized settings for screen components (menu, toolboxes, tables, portal, calendar, etc.), 
* System settings (notifications, email delivery, character set, etc.) 
* as well as the settings for restoring the last session.

###  Tab WebDAV {: #webdav}

The tab "WebDAV" provides you with the [WebDAV link to your OpenOlat system](../basic_concepts/Using_WebDAV.md), which can be used to conveniently manage files. As an OpenOlat user, you can access your personal folder via WebDAV. OpenOlat authors can organize their entire course files via WebDAV.
