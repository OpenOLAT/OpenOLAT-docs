# Anonyme Gäste und externe Benutzer:innen {: #guests_and_externals}

OpenOlat kann neben registrierten Benutzern auch benutzt werden von

* **Gästen** (anonym, können nur die für Gäste frei gegebenen Angebote nutzen)
* **externen Benutzer:innen** (sind mindestens mit Mail bekannt, sind per Mail nur in einzelne, ganz bestimmte Kurse/Gruppen/Portfolios/Projekte eingeladen, maximale Nutzungsdauer: Voreinstellung 180 Tage)

Administrator:innen können festlegen, ob und in welchem Umfang OpenOlat so genutzt werden kann.

## Anonyme Gäste {: #guests}
  
![login_anonymous_external_user_tab1_v1_de.png](assets/login_anonymous_external_user_tab1_v1_de.png){ class="shadow lightbox" }

* **"Gastlogin auf Login Seite":**   
Bedeutet, dass der Gastzugang auf der Login-Seite ausgewählt werden kann.

*  **"Links für Gast":**   
In der Zugangskonfiguration können so Kurse auch an Gäste freigegeben werden.

*  **"Volltextsuche für Gast":**   
Ermöglicht es Gästen, für sie freigeschaltete Kurse systemweit zu durchsuchen.

## Externe Benutzer:innen {: #externals}

![login_anonymous_external_user_tab2_v1_de.png](assets/login_anonymous_external_user_tab2_v1_de.png){ class="shadow lightbox" }

### Kurs

**"Einladung externe Benutzer:innen":** <br> 
Hier wird aktiviert, ob externe Benutzer:innen grundsätzlich für Kurse zugelassen sind und eingeladen werden können. 

**Rolle kann einladen**<br>
Hier wird bestimmt, von welchen Rollen externe Benutzer:innen in Kurse eingeladen werden dürfen:<br>
- von Administrator:innen<br>
- von Lernressourcenverwalter:innen

**Besitzer:in mit Autorenrecht kann einladen**<br>
- **Für alle Kurse aktivieren:** Bedeutet, dass Kursbesitzer:innen externe Benutzer:innen jederzeit einladen können.<br>
- **Muss pro Kurs aktiviert werden:** Bedeutet, dass eine Administrator:in oder Lernressourcenverwalter:in die Funktion in den Kurseinstellungen zuerst aktivieren muss, bevor Kursbesitzer:innen externe Benutzer:innen in einen Kurs einladen können.

### Gruppen

**"Einladung externe Benutzer:innen":**   
Hier wird aktiviert, ob externe Benutzer:innen grundsätzlich im Bereich "Gruppen" zugelassen sind und eingeladen werden können. 

**Rolle kann einladen**<br>
Hier wird bestimmt, von welchen Rollen externe Benutzer:innen in Gruppen eingeladen werden dürfen:<br>
- von Administrator:innen<br>
- von Gruppenverwalter:innen

**Betreuer:in mit Autorenrecht kann einladen**<br>
- **Für alle Gruppen aktivieren:** Bedeutet, dass Gruppenbetreuer:innen externe Benutzer:innen jederzeit einladen können.<br>
- **Muss pro Gruppe aktiviert werden:** Bedeutet, dass eine Administrator:in oder Gruppenverwalter:in die Funktion in der Gruppenadministration zuerst aktivieren muss, bevor Gruppenbetreuer:innen mit Autorenrechten externe Benutzer:innen in einen Kurs einladen können.

### Portfolio

**"Einladung externe Benutzer:innen":**   
Hier wird aktiviert, ob externe Benutzer:innen grundsätzlich im Bereich "Portfolio" zugelassen sind und eingeladen werden können. 


### Projekt

**"Einladung externe Benutzer:innen":**   
Hier wird aktiviert, ob externe Benutzer:innen grundsätzlich in Projekten zugelassen sind und eingeladen werden können. 


!!! tip "Tipp"

    In der Benutzerverwaltung unter **Kontotyp > Externe Konten** kann man sich einen Überblick über alle externen Benutzer:innen verschaffen.
