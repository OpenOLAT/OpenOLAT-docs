# Glossary

A reference glossary of product-specific terms used in OpenOlat. These are terms that have specific meaning within the OpenOlat LMS context.

### Absence management

Module for tracking and managing participant absences from lectures.

### Absence manager

Can manage attendance tracking and absence records across the system.

### Access code

A password-based booking method restricting course enrolment to code holders.

### Access control

Booking and offer management for learning resources.

### Access for coach

Key figure label in the Course widget for courses with the publication status "Access for coach".

### Administration

The system administration area for configuring modules, managing users, and system settings.

### Administrator

Has full system administration rights.

### Adobe Connect

A course element (building block) that can be added to a course structure.

### AI feature

An AI-powered area of use in OpenOlat (e.g. MC Question Generator, Image Description Generator) to which an AI provider and a model are assigned in the AI module.

### AI module

Integration of AI capabilities for content generation and assistance.

### AI provider

The service provider whose AI models are used for the AI-powered features in OpenOlat. Multiple providers can be configured in parallel in the AI module (OpenAI, Anthropic Claude, generic OpenAI-compatible services).

### All correct answers

Scoring method for multiple-choice, matrix, and drag&drop questions: the full score is only achieved if all correct answers are selected, otherwise the final score is 0.

### Animation

A type of learning resource that can be created or imported in the authoring area.

### Appointment scheduling

A course element (building block) that can be added to a course structure.

### Appointments widget

Widget on the Dashboard showing upcoming appointments from the Lectures and Absences module.

### Assessment

A course element (building block) that can be added to a course structure.

### Assessment inspection

Supervised review of completed assessments.

### Assessment management

Central overview and management of assessments.

### Assessment mode

Controlled exam environment with restricted browser access.

### Assessment tool

Central interface for coaches and owners to evaluate, grade, and manage learner assessments.

### Assignment of dates

A course element (building block) that can be added to a course structure.

### Audio

A type of learning resource that can be created or imported in the authoring area.

### Audio/Video recording

Recording capabilities for audio and video content.

### Author

Can create and manage learning resources.

### Authoring

The area where authors create and manage learning resources.

### Badges

Digital badges awarded for achievements.

### BigBlueButton

Web conferencing integration for virtual classrooms.

### BigBlueButton

A course element (building block) that can be added to a course structure.

### Blog

A course element (building block) that can be added to a course structure.

### Blog

A type of learning resource for blog content.

### Blog entry

A type of learning resource that can be created or imported in the authoring area.

### Booking order

A record created when a user books or enrolls in a course through the catalog.

### Bulk assessment

Feature allowing coaches to assess multiple participants simultaneously.

### Calendar

Personal and course calendar for scheduling and event management.

### Calendar

A course element for embedding a course calendar with events and deadlines.

### card2brain

card2brain flashcard learning integration.

### card2brain

A course element for flashcard-based learning via card2brain.

### Catalog

Structured catalog for publishing and finding learning resources.

### Catalog widget

Widget on the Dashboard showing catalog offers from a Course Planner implementation.

### Certificates

Automatic or manual PDF certificate generation.

### Certification program

A structured certification path with recertification requirements.

### Check list

A course element (building block) that can be added to a course structure.

### Check list (old)

A course element (building block) that can be added to a course structure.

### Coach

Tutors/coaches participants in a course or group.

### Coach folder

A dedicated file area visible only to coaches within a course.

### Coaching

Cross-course overview for coaches to monitor learners. See also: Coaching Tool.

### Coaching Tool

The name used in OpenOlat for the tool in which coaches view learners, test corrections, and assessments across courses. See also: Coaching.

### Competence

Skill or competence assigned to users via taxonomy.

### Contact tracing

Location-based contact tracing system.

### Content editor

WYSIWYG editor for creating rich content within Page course elements.

### Conventional course

Traditional course design with menu-based navigation (non-learning-path).

### Corrector

A person assigned to manually grade test submissions within a correction workflow.

### Course

The main learning resource type for structured online courses.

### Course content widget

Widget on the Dashboard showing course contents of a Course Planner implementation.

### Course editor

Tool used to build and modify course structure and course elements.

### Course element

A building block that can be added to a course structure in the course editor.

### Course lifecycle

The automated process by which a course transitions through statuses based on its execution end date.

### Course planner

Can manage curricula and products in the Course Planner module.

### Course Planner

Manage curricula, implementations, and educational structures (formerly "Curriculum").

### Course reminders

Automated email reminders based on course conditions.

### Course template

A course template in the Course Planner that can be instantiated as a course for a curriculum implementation at a later date. See also: Template course.

### Course widget

Widget on the Dashboard of the Coaching tool, showing the courses coached by the current coach.

### Courses

The main area where users find and access their enrolled courses.

### CP learning content

A course element (building block) that can be added to a course structure.

### CP learning content

A type of learning resource for IMS Content Packages.

### Credit points

System for awarding and tracking academic credit points for course completion.

### Curriculum element

A node in the curriculum tree representing a module, semester, or subject.

### Dashboard

Personal start page with configurable widgets.

### Data collection

Surveys used within the Quality Management module to gather feedback.

### Data collection generator

Automated rule sets that create data collections (surveys) based on triggers like course start or end.

### Delivery format

### Document
