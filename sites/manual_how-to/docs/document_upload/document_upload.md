# How do I set up document submission options? {: #document_upload}


??? abstract "Purpose and scope of this guide"

    The overview below is intended to give you ideas about the file upload options available in OpenOlat.

??? abstract "Target audience"

    [x] Authors [x] Coaches  [ ] Participants

    [x] Beginners [x] Advanced  [ ] Experts


??? abstract "Expected prior knowledge"

    * You have already created an OpenOlat course.


## Overview of options {: #overview}

There are various options for where and how course participants can upload a file (e.g. the solution to a task or an essay). Let the following list inspire you:

* [Course Element Participant Folder >](../../manual_user/learningresources/Course_Element_Participant_Folder.md) (Folder accessible only to the individual participant and coaches)
* [Course Element Folder >](../../manual_user/learningresources/Course_Element_Folder.md) (Folder contents visible to all course participants. Upload by participants only if permitted: in the course editor "Tab: Folder configuration > Section: User rights")
* [Course Element Task >](../../manual_user/learningresources/Course_Element_Task.md)
* [Course Element Grouptask >](../../manual_user/learningresources/Course_Element_Grouptask.md)
* [Course Element Portfolio Task >](../../manual_user/learningresources/Course_Element_Portfolio_Task.md)
* [Course Element E-mail >](../../manual_user/learningresources/Course_Element_Email.md) (E.g. responsible coaches configured as recipients, file as attachment)
* [Course Element File Dialog >](../../manual_user/learningresources/Course_Element_File_Dialog.md) (If upload by participants has been permitted.)
* [Course Element Forum >](../../manual_user/learningresources/Course_Element_Forum.md) (If permitted: files as attachments to forum posts)

### Comparison {: #comparison}

| | **Task / Grouptask** | **Participant Folder** | **Folder** |
|---|---|---|---|
| Integrated workflow with task selection, submission deadline, feedback, assessment | Yes | No | No |
| Notifications | Yes | No | No |
| Submission deadline | Yes | Yes | No |
| Reusable for multiple submissions | No (Uploading multiple documents for the same submission deadline is possible.) | Yes | Yes |
| Subfolders | No | Yes | Yes |
| Set max./min. number of documents | Yes | Max. only | No |
| Feedback to participants | Yes | Upload document to "coach return folder" | No (by e-mail only) |
| Revision of submission with deadline | Yes | No | No |
| Automatic assignment of tasks (with multiple task definitions) | Yes | No | No |
| Allow late submission | Yes | No | No |
| Automatic assignment of coaches | Yes | No | No |
| Assessment option in course element | Yes | No | No |
| Visibility of uploaded files for participants | Own submission and feedback only | Own submission and feedback only | All participants can see the files in the folder |
