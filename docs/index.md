# OpenOlat Documentation

OpenOlat is a web-based open source learning management system (LMS) for teaching,
learning, assessment and collaboration, developed by [frentix GmbH](https://www.frentix.com).
This site is the official OpenOlat documentation. The current release is **OpenOlat 20.3**;
the manuals below document this version.

[Zur Deutschen Version wechseln](/de/){ .md-button }

## Documentation sections

- [User manual](manual_user/) – working with courses, tests, groups and learning
  resources, for authors, coaches and learners
- [Admin manual](manual_admin/) – configuration, modules, roles and rights, and
  operation of an OpenOlat instance
- [Release notes](release_notes/) – new features and changes per OpenOlat version
- [How to](manual_how-to/) – step-by-step guides for common tasks
- [Developer cookbook](manual_dev/) – REST API, theming, documentation guidelines
  and development
- [Reference glossary](reference_glossary/) – definitions of OpenOlat terms
