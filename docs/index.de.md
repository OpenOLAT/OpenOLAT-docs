# OpenOlat Dokumentation

OpenOlat ist ein webbasiertes Open Source Lernmanagementsystem (LMS) für Lehren,
Lernen, Prüfen und Zusammenarbeiten, entwickelt von der [frentix GmbH](https://www.frentix.com).
Diese Website ist die offizielle OpenOlat Dokumentation. Das aktuelle Release ist
**OpenOlat 20.3**; die folgenden Handbücher dokumentieren diese Version.

[Change to English version](/){ .md-button }

## Bereiche der Dokumentation

- [Benutzerhandbuch](manual_user/) – Arbeiten mit Kursen, Tests, Gruppen und
  Lernressourcen, für Autoren, Betreuer und Lernende
- [Administrationshandbuch](manual_admin/) – Konfiguration, Module, Rollen und
  Rechte sowie Betrieb einer OpenOlat-Instanz
- [Release Notes](release_notes/) – neue Funktionen und Änderungen pro
  OpenOlat-Version
- [How-to](manual_how-to/) – Schritt-für-Schritt-Anleitungen für häufige Aufgaben
- [Developer Cookbook](manual_dev/) – REST API, Theming, Dokumentationsrichtlinien
  und Entwicklung
- [Referenzglossar](reference_glossary/) – Definitionen der OpenOlat-Begriffe
